01

抵達日

7月7日（二）
公園 → 展望台 → 午餐 → Check-in

📍 機場 → Banna Park → Southern Island Observation Deck → やえかつ → 牛乳屋さん → MaxValu Yaima → 飯店Check-in → 燒肉樹苑

在 Google 地圖開啟今日路線
08:35
抵達新石垣機場
出關、領行李，預估約 30–40 分鐘，接著機場取車。
09:30–10:30
Banna Park
日本最南端的森林公園，有觀景台與大型遊樂設施。
選擇考量：飯店到市區路上順路，早上較涼爽，適合抵達當天讓小朋友先活動筋骨，不需走完全部五區，1小時走主要觀景台即可。
10:40–11:10
Southern Island Observation Deck
獨特造型的展望台，可俯瞰島嶼大半景觀，免費停車。
選擇考量：與Banna Park距離僅約1公里、緯度幾乎相同，動線完全順路；展望台造型特別好拍照，停留時間短，不會拖累行程。
11:30–13:00
🍽️ やえかつ（午餐）
石垣島人氣炸豬排專門店，里肌與腰內肉炸得酥脆多汁，肉質嫩得像和牛。
選擇考量：評價4.5、全週無休11:00開始營業；位於市區公設市場旁，與當天動線完全吻合。
13:30–14:15
🥛 石垣島の牛乳屋さん
自家牧場鮮奶製作的奶昔、布丁、優格，店內有可愛乳牛裝飾與牛牛造型椅。
選擇考量：15:00才能check-in，這裡有冷氣可休息，當作入住前的緩衝點，小朋友喜歡店裡的牛牛裝飾。
14:30–15:00
MaxValu Yaima（超市）
24小時營業大型超市，生鮮、零食、日用品一次買齊。
選擇考量：採買飯店期間零食、水、防曬乳，安排在check-in前最後一站，買完直接帶進房間放，動線最順。
15:00
🛏️ ANA InterContinental Check-in
正式入住，安頓行李。
15:30–18:30
🏖️ 飯店設施 ・ 自由活動
游泳池、沙灘散步或房內休息，第一天不安排外出，留時間給家人放鬆。
選擇考量：Check-in、採買都結束後，讓大家有充裕時間整理行李、熟悉飯店環境，不用再趕行程。
19:00 起
🍖 石垣島燒肉樹苑（晚餐）
市區人氣燒肉店，肉質軟嫩、份量實在。
選擇考量：抵達第一天需要補充體力，燒肉份量足、可單點各取所需，適合長輩小孩同桌不挑食。
02

北部探索日

7月8日（三）
海灘 → 玻璃船 → 購物

📍 Sukuji Beach → 川平灣玻璃船 → 三川食堂 → 飯店午休 → 公設市場 → Euglena Mall → THE NORTH FACE → Sora → 晚餐

在 Google 地圖開啟今日路線
09:00
飯店出發
早餐後出發，往北部前進，車程約45分鐘。
09:50–11:20
Sukuji Beach 戲水
海浪平緩、淺灘範圍寬廣，是家庭與初學者的人氣海灘。
選擇考量：退潮時水位更淺，適合幼童戲水；有遮蔭樹木與淋浴設施。早上較涼爽，安排在玻璃船之前，把較曬的時段留給有遮頂的船。
幼童友善・有遮蔭
11:30–12:15
川平灣玻璃船
米其林三星景點，全程約30分鐘，0歲起可參加，不需下水。
選擇考量：現場購票即可，每15分鐘一班不需預約。建議選有冷氣的Blue Coral船班，夏天較舒適。船票：大人約¥1,300-1,700、小孩約¥650-800。
全齡友善・免下水
12:30–13:30
🍽️ 三川食堂（午餐）
在地人氣食堂，炸魚定食、套餐選擇多。
選擇考量：位置剛好在川平灣回飯店的路上，不需繞路。備用選項：公園茶屋（川平灣旁，1961年老字號八重山蕎麥麵）。
😴
午休時段 13:45–15:30　小朋友返回飯店房間補眠休息。
16:00–18:00
市區購物：公設市場 ＋ Euglena Mall ＋ THE NORTH FACE ＋ Sora
公設市場逛伴手禮小吃，Euglena Mall 商店街，接著到 THE NORTH FACE 石垣（石垣限定商品）與樓上的 Sora（戶外品牌 select shop）。
選擇考量：四個地點都在市區同一帶，步行串連不需移車；THE NORTH FACE 與 Sora 同棟樓（1F＋2F），一次逛完。
18:30 起
🍽️ 晚餐：あじまー商店 或 Eiko Seafood Izakaya
兩個選項都在市區，沖繩家庭料理 / 道地海鮮居酒屋。
選擇考量：あじまー商店有英文菜單QR code，溝通較方便；Eiko Seafood Izakaya評價極高、魚貨新鮮，⚠️ 建議提前訂位（深受當地人喜愛，熱門時段容易客滿）。
03

西部海灘日

7月9日（四）
海灘 → 冰淇淋 → 午餐 → 泳池 → 🎂 烤肉慶生

📍 富崎海灘 → MiruMiru本舗 → Kabusa食堂 → 飯店泳池 → 花園烤肉區

在 Google 地圖開啟今日路線
09:30
飯店出發
今天往西北方向，車程較短。
10:10–11:10
富崎海灘
白色棧橋延伸入海，被稱為「天使碼頭」，沙質乾淨柔軟。
選擇考量：免費入場（穿過飯店即可進入），海水清澈、適合拍全家福，跟MiruMiru本舗距離很近、動線順。
11:15–12:00
MiruMiru本舗
自家牧場鮮奶製作的義式冰淇淋，可俯瞰名蔵灣海景。
選擇考量：位於高處露台，能邊吃冰淇淋邊賞海景，與富崎海灘距離僅1公里內，順路休息點。
12:15–13:15
🍽️ Kabusa食堂（午餐）
漁港附近食堂，使用當地漁獲製作定食，薑黃手打蕎麥麵是招牌。
選擇考量：位置剛好在回程路上、全週無公休（原本想去的Kunatsuyu週四公休，故改選此處），份量大、適合長輩的清淡口味。
13:30–17:30
🏊 花園烤肉區專屬泳池玩耍
使用花園烤肉區的專屬泳池，晚餐前盡情戲水放鬆。
選擇考量：晚餐在同一區域，泳池玩耍完直接步行至烤肉區，不需換裝出門，動線最順。
18:30
🎂 花園烤肉區（生日晚餐）
ANA InterContinental 飯店內戶外花園烤肉，融合石垣島食材的烤肉饗宴。
選擇考量：飯店內烤肉區，不需外出移動；有專屬泳池，小孩大人都開心，生日氣氛輕鬆歡樂。
04

東北景觀＋購物日

7月10日（五）
展望台 → 唐吉訶德 → 燒肉收尾

📍 玉取崎展望台 → Ishigaki Market → 飯店午休 → 空擋 → Ishigaki beef MARU

在 Google 地圖開啟今日路線
09:00
飯店出發
往東北方向，車程約30-40分鐘。
09:40–10:30
玉取崎展望台
可眺望伊原間灣藍色海水與珊瑚礁、Hanna山、平久保半島。
選擇考量：原本考慮平久保埼燈塔，但來回車程超過2.5小時太遠。玉取崎是必經之路、步道平緩，穿拖鞋也OK，適合長輩小孩，不用走到最北端也能看到開闊海景。
10:45–12:00
🍽️ Ishigaki Market（早午餐）
蒜香鮮蝦飯、舒芙蕾鬆餅評價極高，環境舒適、自然採光。
選擇考量：評價4.6全島數一數二，鬆餅現點現做，剛好接續展望台行程，作為北部一帶的補給站。
😴
午休時段 12:45–15:00　返回飯店休息，小朋友可在車程上補眠。
15:00–18:30
🕐 空擋時間
依當天狀況彈性安排，可選擇回飯店休息、使用飯店設施，或到附近購物（唐吉訶德距飯店約5分鐘車程）。
19:00 起
🍖 Ishigaki beef MARU（晚餐）
7種牛肉套餐、內臟鮮嫩，少數可刷卡的餐廳之一。
選擇考量：評價4.5、有英文菜單，最後一晚以燒肉收尾。⚠️ 建議提前訂位，旺季容易客滿。

早起・還車・搭機

07:30–08:00
前往機場還車
